# Crypto Currency Market Updates Microservice

This microservice app provides a convenient way to fetch crypto currency market updates from the Bittrex API.

## Features

- Retrieve market summaries for all cryptocurrencies
- Get market summary for a specific cryptocurrency

## Prerequisites

Before running the microservice, ensure that you have the following prerequisites installed:

- Python 3.10 or higher
- Pip package manager
- Additional python libraries mentioned in requirements.txt


## Installation

1. Clone the repository:

   ```shell
   git clone https://github.com/psuresh-pydev/crypto-currency-market-updates.git
   ```

2. Change to the project directory:

   ```shell
   cd crypto-currency-market-updates
   ```

3. Install the required dependencies:

   ```shell
   pip install -r requirements.txt
   ```


## Usage

Start the microservice by running the following command:

```shell
cd market_microservice
python main.py
```

The microservice will start running on `http://localhost:5000`.

### Endpoints
- **GET /**: View API docs generated by swagger
- **GET /markets/summaries**: Get all market summaries
- **GET /markets?market=<symbol>**: Get market summary for a specific cryptocurrency (e.g., /markets?market=ltc-btc)


## Testing

To run the unit tests for the microservice, execute the following command from project root directory:

```shell
pytest
```

The tests are located in the `tests/` directory and ensure the correct functionality of the microservice.
